# Trial Status Background

## Overall status
ClinicalTrials.gov includes an "overall status" field describing the trial’s recruitment/state (e.g., Recruiting, Active, Completed, Terminated).

## Common interpretation cautions
- Status can change over time; a snapshot reflects the latest posting at time of pull.
- Some statuses are rare or inconsistently used depending on sponsor reporting.

## Implication for analysis
Status is treated as a primary stratification variable alongside phase to describe the recent trial landscape and to contextualize enrollment distributions and results availability.
