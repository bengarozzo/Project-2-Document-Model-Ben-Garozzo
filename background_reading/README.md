# Background Readings

This folder contains short notes that support the domain exposition and document-model decisions for the ClinicalTrials.gov secondary dataset. These notes are not raw data; they summarize external references and definitions relevant to interpreting clinical trial registry fields (phase, status, enrollment, sponsors, and results/outcomes reporting).

Files:
- `01_clinicaltrials_api_notes.md`
- `02_trial_phase_background.md`
- `03_trial_status_background.md`
- `04_enrollment_field_notes.md`
- `05_registry_bias_limitations.md`
